Abi's Colour Match Game

00:00

Match the target colour by adjusting the RGB sliders. RGB colour values for each of red, green and blue range from 0 to 255. The objective is to match the colour in the shortest time possible. The difficulty level determines the tolerance allowed for matching the colour:

  • Expert: The colour must be matched exactly.
  • Medium: The colour must be within 20 units of the target values for each RGB component.
  • Easy: The colour must be within 50 units of the target values for each RGB component.
